# Release Notes

## What's New

- Added class CustomLogger - can be used in place of create_logger.  The CustomLogger class inherits the logging.Logger class so it can be used as a direct replacement.  The CustomLogger class supports updating the console and file level, and includes a background thread that will run to cleanup old files.
- Added "update_console_level" and "update_file_level" functions to allow easy update to the logging level for those using the "create_logger" method to create the logging.Logger object instead of the new class.
